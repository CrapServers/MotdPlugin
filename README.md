# MotdPlugin
Spigot plugin that changes the motd of a minecraft server when it is pinged in the server browser

MOTDs are put in `plugins/MotdPlugin/config.yml`

## Example config

```
#
motds:
- Hey,
- trollhd
- Hello, Your Computer Has Virus
- HI
- §epopbob_ has joined the game.
- §4End It.
```
